⚡ Deploy Automático

Execute nosso script para preparar tudo automaticamente!

# Windows PowerShell
.\deploy.ps1

O script prepara Git, cria commit e exibe instruções personalizadas!

📋 Passo a Passo Completo

1

Enviar para GitHub

Execute os comandos ou use o script deploy.ps1:

git init
git add .
git commit -m "Deploy to production"
git remote add origin URL_DO_SEU_REPO
git push -u origin main
2

Escolher Plataforma

Recomendamos Vercel pela simplicidade. Acesse vercel.com e faça login com GitHub.

3

Importar Projeto

Clique em "Add New Project" → Selecione o repositório → Deploy! Em 1-2 minutos seu site estará no ar.

4

Configurar Domínio (Opcional)

Se você tem um domínio próprio, adicione nas configurações da plataforma. HTTPS será configurado automaticamente!

5

✅ Pronto!

Seu site está no ar com HTTPS! Compartilhe a URL e acompanhe analytics.
